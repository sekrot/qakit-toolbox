# QAKit Toolbox — Политика конфиденциальности

**Обновлено:** 2026-06-05

QAKit Toolbox — это расширение Chrome с набором офлайн-утилит для
разработчиков и QA-инженеров (JSON, JWT, regex, хэши, timestamps, UUID,
base64/URL, сравнение текстов, история буфера обмена, подбор цвета,
снимки экрана с аннотациями).

## Если коротко

**Мы не собираем, не передаём и не продаём ваши персональные данные.
Никогда.**

Всё, что делает QAKit Toolbox, происходит локально на вашем устройстве.
Никакие данные не уходят на наши серверы — у нас просто нет серверов.

## Какие данные обрабатываются

Расширение читает и пишет данные **только на вашем устройстве** через
`chrome.storage.local` — изолированное хранилище, доступное только этому
расширению.

| Данные                                                    | Зачем                                         | Хранится      |
| --------------------------------------------------------- | --------------------------------------------- | ------------- |
| Ваши настройки (тема, язык, флаг знакомства)              | Сохранение предпочтений UI                    | На устройстве |
| История буфера обмена, которую вы сохраняете в side panel | Утилита «Сохранение буфера обмена»            | На устройстве |
| История подбора цвета                                     | Утилита «Подбор цвета»                        | На устройстве |
| Локально сгенерированный install id и счётчик сессий      | Внутренняя локальная статистика использования | На устройстве |
| Скриншоты страниц, сделанные в утилите «Скриншот»         | Для аннотирования и скачивания                | На устройстве |

Мы **не**:

- отправляем никаких данных на удалённые серверы;
- используем аналитику, рекламу или fingerprinting-библиотеки;
- отслеживаем историю браузера, вкладки или содержимое страниц;
- читаем содержимое веб-страниц (за исключением случая, когда вы сами
  нажимаете «Сделать скриншот» в утилите «Скриншот» — тогда
  используется Chrome API `captureVisibleTab` для одного PNG-снимка
  видимой части активной вкладки по вашему запросу);
- передаём данные третьим лицам — третьих лиц нет.

## Запрашиваемые разрешения и почему

- **`storage`** — для локального хранения настроек и истории.
- **`sidePanel`** — для отображения UI в side panel Chrome.
- **`activeTab`** + **`<all_urls>`** (host permission) — используется
  только для `chrome.tabs.captureVisibleTab` в утилите «Скриншот», когда
  вы нажимаете «Сделать скриншот». Никакие скрипты в страницы не
  внедряются, содержимое страниц не читается и не передаётся.
- **`clipboardRead`** / **`clipboardWrite`** — для чтения буфера обмена
  при нажатии «Сохранить буфер» в side panel и для копирования
  результатов работы утилит (JSON, хэши, UUID и т. п.) обратно в буфер.

## Экспорт и удаление данных

- **Экспорт** — _Настройки → Данные → Экспорт настроек_ скачает JSON-файл
  со всем, что хранится локально.
- **Удаление** — _Настройки → Данные → Очистить историю_ удалит историю
  буфера обмена и подбора цвета. Удаление расширения убирает всё, что
  Chrome хранит для этого расширения, включая локальное хранилище.

## Дети

QAKit Toolbox — инструмент для разработчиков, он не предназначен для
детей младше 13 лет. Мы сознательно не собираем данные ни о ком,
включая детей.

## Изменения политики

Если эта политика изменится, новая версия будет опубликована по тому же
адресу с обновлённой датой «Обновлено». О существенных изменениях будет
сообщено в release notes на странице расширения в Chrome Web Store.
